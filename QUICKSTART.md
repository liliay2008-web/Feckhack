# Быстрый старт

## 1. Установка зависимостей

```bash
npm run install:all
```

## 2. Настройка окружения

Создайте файл `backend/.env`:
```env
PORT=3001
JWT_SECRET=your-secret-key-change-in-production
JWT_EXPIRES_IN=7d
```

## 3. Запуск

```bash
npm run dev
```

## 4. Откройте браузер

Перейдите на http://localhost:3000

## 5. Первые шаги

1. Зарегистрируйтесь как **Администратор**
2. Загрузите студентов (раздел "Загрузка студентов")
3. Загрузите темы (раздел "Загрузка тем")
4. Распределите темы (раздел "Распределение тем")

## Формат файлов

### Студенты (Excel/CSV)
Колонки:
- `ФИО` или `ФИО студента` или `full_name`
- `Группа` или `group_name` или `Группа студента`
- `Телефон` или `phone` (опционально)

### Темы (Excel/CSV)
Колонки:
- `Название` или `Тема` или `title` или `Название темы`
- `Руководитель` или `supervisor` (опционально)

## Особенности

- **Блокировка тем**: Когда староста выбирает тему, она блокируется на 30 минут
- **Автоматическое распределение**: Администратор может распределить темы автоматически
- **Экспорт**: Данные можно экспортировать в Excel, CSV или PDF
- **История изменений**: Все действия записываются в историю
